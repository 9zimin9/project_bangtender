# project_bangtender

## 프로젝트 소개

### [방텐더 (project_bangtender)]
Django 와 Django REST Framework를 기반으로 사용자 맞춤 칵테일 추천 및 주류 정보를 제공하는 웹 애플리케이션(서비스@임시)입니다. 사용자 맞춤 칵테일, 양주 추천 기능과 주류 매장 찾기 등의 기능을 통해 칵테일을 좋아하는 사용자에게 유용한 정보를 제공합니다.

## 프로젝트 개요

### 프로젝트 개발 배경
코로나 이후로 회식이 적어진 국내 사회에서 20~30대 소비층에 국내의 주요 소비 주류인 소주나 맥주가 아닌 양주의 인기가 커지고 있다는 정보를 보게 되었고, 양주의 정보 제공과 다양한 칵테일 제조 방법을 추천 및 제공해 주는 플랫폼이 있으면 좋겠다고 생각하였습니다. 이와 같은 배경으로 사용자의 선호 취향을 토대로 칵테일 및 양주를 추천해 주는 OPENAI를 이용한 방텐더 봇, 지도 API를 통해 사용자 주변의 주류 매장을 보여주는 기능과 함께 주류나 칵테일에 대한 정보를 같이 제공할 수 있는 웹 사이트를 생각하게 되었습니다.

### 프로젝트 기대 효과
서비스를 이용하는 유저가 주류나 칵테일에 대한 정보가 부족하더라고 쉽게 정보를 찾을 수 있으며, 자신의 기호에 맞는 주류와 칵테일을 방텐더 봇을 이용하여 추천받으며 잘 알지 못하는 주류에 대한 정보나 구매하기에 비싼 가격의 벽에 막혀 접하기 힘들었던 부분에 대해 다양한 정보를 제공하여 주류에 대해 쉽고 친근하게 접근할 수 있을 것입니다.
또한 어필리에이트 마케팅을 통해 제휴를 맺은 온라인 스토어로 연결되는 어필리에이트 링크를 제공하거나 주류 관련 브랜드들과 제휴를 맺어, 웹사이트에 배너 광고나 스폰서쉽을 유치하는 등의 서비스를 제공하고, 지역 주류 매장 검색 시 특정 매장을 상단에 노출하는 유료 광고나 지역 주류 매장에서의 주문 서비스를 제공해 수수료를 제공받아 수익화를 기대해 볼 수 있습니다.

## 기능적 요구사항

### 1. 페이지

#### 1.1 메인 페이지

- **랜덤 칵테일 레시피**: 모든 사용자에게 일별로 무작위 칵테일 레시피 제공
- **맞춤 칵테일 추천**: 회원에게만 제공, 프로필 정보에 따라 유저 맞춤형 추천 제공
- **Info 제공**: 칵테일 및 주류 관련 팁, 역사, 재미있는 이야기를 제공

#### 1.2 상단 네비게이션 바

- **로고**: 메인 페이지로 이동
- **주류 페이지**: 다양한 주류 정보를 제공하는 페이지로 이동
- **칵테일 페이지**: 다양한 칵테일 정보를 제공하는 페이지로 이동
- **OpenAI 추천받기**: 회원에게만 제공되는 칵테일 추천 기능
- **주변 주류 매장 찾기**: 회원의 주소 기반으로 근처 주류 매장 검색
- **마이 프로필**: 회원 정보 및 설정 관리 페이지로 이동

#### 1.3 주류 페이지

- **목록 페이지**: 주류의 이름, 가격, 이미지 등 정보 제공
- **상세 페이지**: 선택한 주류의 맛, 도수, 가격 등 상세 정보 제공

#### 1.4 칵테일 페이지

- **목록 페이지**: 칵테일의 이름, 이미지 등 정보 제공
- **상세 페이지**: 선택한 칵테일의 레시피, 맛, 만드는 법 등 상세 정보 제공

#### 1.5 마이 프로필

- 회원 정보 조회 및 수정
- 북마크한 술과 칵테일 페이지로 이동 가능
- 회원 탈퇴 기능 제공

#### 1.6 로그인 및 회원가입 페이지

- **로그인 페이지**: JWT 발급을 통한 사용자 인증
- **회원가입 페이지**: 유저 정보 등록 및 인증 처리

### 2. 회원별 기능 차별화

- **비회원**: 랜덤 칵테일 레시피 열람 가능 >> 일별로 제공되는 랜덤 칵테일 레시피 제공
- **회원**: 모든 기능 접근 가능, 맞춤 칵테일 추천 및 주류 매장 찾기 제공

### 3. 기능 및 모델 정의

- **랜덤 칵테일 레시피 제공**: 데이터베이스에서 무작위로 칵테일 레시피를 가져와 메인 페이지에 표시 > + 매일 다른 레시피 제공
- **사용자 맞춤 칵테일 추천**: 사용자의 선호도에 기반하여 추천 제공
- **Bangtenderbot 기능**: 맞춤형 추천 제공
